About


                
 JSON Schema Lint is a JSON schema validator to help you write and test JSON Schemas that conform with various specification versions.
                    
The author/maintainer is Nick Maynard. Fork this project on Github.

                    
Libraries:
                        
    
                            
  • Gary Court's JSV :: draft-01, draft-02, draft-03
    • 
                            
  • Evgeny Poberezkin's Ajv :: draft-04, draft-05 (partial)
    • 
                        

                    

                    
Optionally, you may use schemas and documents in the YAML format. These documents are parsed with Jérémy Fairve's yaml.js library.
